
The objective of the assignment was to analyze global pollution data and explore how environmental indicators relate to energy recovery. The project combines both unsupervised and supervised machine learning techniques to understand patterns in pollution and evaluate the ability to predict energy recovery levels.

The assignment was structured into four main phases:

1. **Data Preprocessing & Feature Engineering**  
   - Data cleaning and handling missing values  
   - Feature transformations and preparation for modeling  

2. **Clustering Analysis**  
   - K-Means clustering  
   - Hierarchical clustering  
   - Evaluation using silhouette scores  
   - Identification of environmental grouping patterns  

3. **Energy Recovery Prediction**  
   - Linear regression (baseline model)  
   - Neural network (MLP Regressor)  
   - Hyperparameter tuning  
   - Ensemble models (Random Forest & Gradient Boosting)  
   - Model evaluation using R², MSE, and MAE  

4. **Model Evaluation & Insights**  
   - Comparison of clustering and regression results  
   - Interpretation of model performance  
   - Discussion of environmental and policy implications  

This project demonstrates a complete machine learning workflow, including preprocessing, modeling, evaluation, and critical interpretation of results.
